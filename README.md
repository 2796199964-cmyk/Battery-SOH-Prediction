## 📖 项目简介

**本项目** 是一套面向新能源汽车电池健康状态（State of Health, SOH）与剩余使用寿命的高精度数据驱动预测系统。

本项目为 **零碳科技解决方案创新大奖赛（妙算榜）** 获奖参赛方案。针对传统电池寿命预测方法误差率高（高达25%）、难以量化动态环境耦合影响等痛点，本项目创新性地提出了一种 **基于经验模态分解（EMD）与 XGBoost-GPR 混合架构** 的预测算法。通过多尺度信号分解与异构模型优势互补，实现了对车载电池复杂老化轨迹的精准、平滑预测。

## 🎯 行业痛点与技术瓶颈

1. **经济与环保双重驱动**：电池健康度直接决定新能源二手车估值（影响占比30%-50%），且精准预测是落实工信部“电池回收率提升至70%”规划的前提。
2. **数据复杂与动态耦合**：电池老化受电压、内阻、温度等多参数非线性耦合影响（如偶发高温暴晒难以量化）。
3. **传统方法局限**：依赖固定物理公式，忽略动态环境因素；而单一深度学习模型计算量大，难以部署且易过拟合。

## 🚀 核心技术架构

本项目将时间序列预测问题转化为监督学习问题，核心流程分为三大模块：

### 1. 数据清洗与特征工程 (Data Processing)
- **工况筛选**：精准截取 SOC 在 **50%~80%** 区间的充电数据。该区间内锂离子活性均衡，充放电行为最稳定，确保容量计算可靠性。
- **变体安时积分法**：结合时间戳与 SOC 变化，精准计算电池实际可用容量 ($C_a$)。
- **中值滤波去噪**：采用非线性中值滤波替代传统均值滤波，在滤除尖峰噪声的同时，完美保留电池容量衰减的边缘特征与动态趋势。
- **滑动窗口特征构建**：利用滑动窗口将历史 SOH 观测值集合化，构建高维时序特征矩阵。

### 2. 核心算法：EMD-XGBoost-GPR 混合预测模型
我们摒弃了单一模型的局限性，设计了 **“分解-分治-重构”** 的混合架构：

1. **EMD (经验模态分解)**：将复杂的非平稳 SOH 衰减信号分解为多个高频噪声分量（IMF）和低频平稳残差。
2. **XGBoost 处理高频噪声**：针对包含突变点和复杂非线性模式的高频 IMF 序列，利用 XGBoost 强大的梯度提升树集成学习能力进行精准拟合。
3. **GPR (高斯过程回归) 处理低频残差**：针对平稳的低频残差序列，利用 GPR 的贝叶斯非参数特性进行建模，不仅保证了曲线的极致平滑性，还能提供**预测置信区间（不确定性估计）**。

### 3. 消融实验与对比验证
通过大量对比实验证实了本架构的优越性：
- **对比单一 GPR**：引入 EMD 分解后，有效滤除了高频噪声，避免了 GPR 直接拟合非平稳信号导致的精度下降。
- **对比单一 XGBoost**：利用 GPR 替代 XGBoost 处理残差序列，避免了树模型在平稳趋势上的局部过拟合与震荡，使预测曲线更符合物理衰减规律。
