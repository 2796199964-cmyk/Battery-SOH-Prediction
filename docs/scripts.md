# 脚本说明

## 数据预处理

- `0精简数据.py`：精简原始数据字段。
- `1筛选.py`：按时间和 SOC 趋势筛选数据。
- `2去除SOC和充电状态异常值.py`：去除 SOC 和充电状态异常数据。

## 特征工程

- `31筛选SOC区间.py`：筛选 SOC 50% 到 80% 区间。
- `32电池容量积分.py`：使用安时积分思路计算容量。
- `3132筛选SOC+积分.py`：合并 SOC 筛选和容量积分步骤。
- `33提取_里程-容量.py`：提取里程与容量关系。
- `34作图_里程-容量.py`：绘制里程-容量关系图。
- `41去除异常值(容量).py`：去除容量异常值。
- `42中值滤波.py`：对容量或 SOH 序列进行中值滤波。
- `5_SOH计算.py`：计算 SOH。

## 模型预测

- `51_GPR.py`：单一 GPR 预测。
- `51_SVR 预测.py`：SVR 预测。
- `51_EMD-GPR.py`：EMD 分解后使用 GPR 预测。
- `51_EMD-SVR 预测.py`：EMD 分解后使用 SVR 预测。
- `51_EMD-LSTM预测.py`：EMD 分解后使用 LSTM 预测。
- `51_EMD-XGboost.py`：EMD 分解后使用 XGBoost 预测。
- `51_EMD-XGboost-GPR.py`：EMD-XGBoost-GPR 组合预测，是答辩方案中的重点模型。
- `61_里程预测.py`、`61_预测两年otometer.py`：里程和未来趋势预测。
- `72_下降模型预测.py`、`81_下降模型预测.py`：下降趋势模型预测。
